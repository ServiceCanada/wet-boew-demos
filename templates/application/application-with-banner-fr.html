Mon application

Page d'application avec bannière informative

Veuillez noter que la présente page web représente une contribution communautaire et qu'elle ne bénéficie pas de la validation ni de l'approbation formelle du Bureau de la transformation numérique comparativement à une conception officielle. Il est à souligner que cette contribution ne va pas à l'encontre d'une version officielle publiée au moment de l'ajout. Si une conception officielle équivalente est élaborée et soit formellement publiée, il est possible que la conception démontrée sur la présente page en diffère en partie ou en totalité.

Cette conception est stable et par conséquent elle bénificie des mêmes avantages que toutes autres fonctionnalités et conceptions stables telles que définies par l'ébauche de l'API publique de version de GCWeb.

Options de configuration (spécifique à GCWeb Jekyll)

  • "infoBanner" (optionnel, Object): contient les propriétés de la bannière d'information.
    • "title" (obligatoire): le titre de la bannière.
    • "message" (obligatoire): le message de la bannière.
    • "link" (optionnel, Object): contient les propriétés du lien.
      • "text" (obligatoire, String): le texte du lien qui s'affiche.
      • "href" (obligatoire, String): l'url du lien qui s'affiche.
      • "external" (optionnel, Boolean): ouvrir le lien dans une nouvelle fenêtre.
    • "button"(optionnel, Object): contient les propriétés du bouton.
      • "text" (obligatoire, String): le texte du bouton qui s'affiche.
      • "href" (obligatoire, String): l'url du bouton qui s'affiche.
      • "external" (optionnel, Boolean): ouvrir le lien dans une nouvelle fenêtre.

Code

{
	...
	"infoBanner": {
		"title": "Version bêta",
		"message": "<strong>Beta version</strong>: We've made some changes to improve your experience with My Service Canada Account.",
		"link": {
			"text": "Learn more about the beta version",
			"href": "https://www.canada.ca/en",
			"external": true
		},
		"button": {
			"text": "Try beta version",
			"href": "https://www.canada.ca/en"
		}
	}
	...
}
