Fonction

(Version 1.0)

Type de fonction Nom Comment implémenter Fonction
Événement jQuery wb-init.dropdown Déclenché manuellement (par exemple : $( ".dropdown" ).trigger( "wb-init.dropdown" );). Initialise la composante .dropdown. Cette composante sera initialisée automatiquement sauf si l'élément .dropdown est ajouté après le chargement de la page et l'initialisation de wet-boew.
Événement jQuery wb-ready.dropdown Déclenché automatiquement après l'initialisation du plugin. Utilisé pour identifier quand et où la composante s'initialise. 
$( document ).on( "wb-ready.dropdown", ".dropdown", function( event ) {
});
$elm.on( "wb-ready.dropdown", function( event ) {
});
