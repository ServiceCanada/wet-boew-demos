Lien pour changer de langue

Statut
Stable
Type
Fonctionnalité du site Canada.ca
Dernière révision
2022-11-01
Version de la composante
Version 1.1

Objectif

Le lien permettant de changer de langue dirige systématiquement la personne vers la page équivalente dans l’autre langue officielle, ce qui fait en sorte qu’il peut passer d’une langue à l’autre sans perdre l’endroit où il se trouvait auparavant.

Comment mettre en œuvre

  • Le lien permettant de changer de langue ne consiste qu’en un seul élément : il s’agit d’un simple lien textuel indiquant « English » ou « Français ».
  • Le lien est toujours affiché dans le coin supérieur droit de l’en-tête général, peu importe la taille de l'écran.
