Gestionnaire JSON avec l'exécuteur d'action de patches

  • Documentation
  • Questions ou commentaires?

Gérer des jeux de données et applique des correctifs JSON avec l'exécuteur d'action de patches.

La valeur est:

Présentement non définie

Code source

<!-- Définition du nom du jeux de données -->
<div id="monIdPourBonjourLeMonde" data-wb-jsonmanager='{"name": "helloworld"}'></div>

<!-- Affichage de la valeur -->
<p>La valeur est:</p>
<div class="mrgn-bttm-lg">
	<output data-json-replace="#[helloworld]/yep">Présentement non définie</output>
</div>

<!-- Boutton d'action -->
<button class="btn btn-default" type="button" data-wb-doaction='{
	"action": "patch",
	"source": "#monIdPourBonjourLeMonde",
	"patches":
		{
			"op": "add",
			"path": "/yep",
			"value": "un test"
		}
	}'>Définir la valeur à: <code>un test</code></button>

		}
