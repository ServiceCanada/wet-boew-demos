Example

Delete Personal Identifiable Information (PII) from the form flagged fields before submit.

View code 
<form action="wb-data-scrub-en.html" >
	<div class="form-content">
		<div class="form-group">
			<label for="preferredNumber">Preferred: <span class="field-name">Phone Number</span></label>
			<input class="form-control" id="preferredNumber" name="preferredNumber" type="tel" data-rule-wbscrub />
		</div>
		<div class="form-group">
			<label for="email"><span class="field-name">Email Address</span> (test@domaine)</label>
			<input class="form-control" id="email" name="email" type="email" data-rule-wbscrub />
		</div>
		<div class="form-group">
			<label for="search"><span class="field-name">Search</span></label>
			<input class="form-control" id="search" name="search" type="search" data-rule-wbscrub />
		</div>
		<div class="form-group">
			<label for="comments"><span class="field-name">Comments</span></label>
			<textarea class="form-control" id="comments" name="comments" data-rule-wbscrub rows="4" cols="50"></textarea>
		</div>
		<div class="form-group">
			<label for="noScrubField"><span class="field-name">No scrub field example</span></label>
			<input class="form-control" id="noScrubField" name="noScrubField" type="text"/>
		</div>
		<button type="submit" class="btn btn-primary">Submit</button>
	</div>
</form>

Test case

Fill the following fields with the following values:

  • Phone: 514 111 1111
  • Email: 1@example.com
  • Search: Passport Nr: AB123456
  • Comments: visa: 4916 0739.3667-6891, master:5428735149026050, SIN:123 123-123, ZIP:K2C 3N2, Driving Licence Nr:P12345678912345, Bank account example: 003-1234567
  • No scrub field example: John Doe

The form will prompt a message informing the user that any Personal Information will not be submitted.

After the form submit, the output of url parameters will be : ?preferredNumber=&email=&search=Passport+Nr%3A+&comments=visa%3A+%2C+master%3A%2C+SIN%3A%2C+ZIP%3A%2C+Driving+Licence+Nr%3AP%2C+Bank+account+example%3A+&noScrubField=John+Doe
