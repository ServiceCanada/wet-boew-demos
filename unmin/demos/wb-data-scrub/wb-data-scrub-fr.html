Exemple

Supprimez les informations personnelles identifiables (IPI) des champs marqués du formulaire avant de les soumettre.

View code 
<form action="wb-data-scrub-fr.html" >
	<div class="form-content">
		<div class="form-group">
			<label for="preferredNumber"><span class="field-name">Numéro de téléphone</span></label>
			<input class="form-control" id="preferredNumber" name="preferredNumber" type="tel" data-rule-wbscrub />
		</div>
		<div class="form-group">
			<label for="email"><span class="field-name">Addresse courriel</span> (test@domaine)</label>
			<input class="form-control" id="email" name="email" type="email" data-rule-wbscrub />
		</div>
		<div class="form-group">
			<label for="search"><span class="field-name">Rechercher</span></label>
			<input class="form-control" id="search" name="search" type="search" data-rule-wbscrub />
		</div>
		<div class="form-group">
			<label for="comments"><span class="field-name">Commentaires</span></label>
			<textarea class="form-control" id="comments" name="comments" data-rule-wbscrub rows="4" cols="50"></textarea>
		</div>
		<div class="form-group">
			<label for="noScrubField"><span class="field-name">Exemple de champ sans IPI</span></label>
			<input class="form-control" id="noScrubField" name="noScrubField" type="text" />
		</div>
		<button type="submit" class="btn btn-primary">Soumettre</button>
	</div>
</form>

Scénario de test

Remplissez les champs suivants avec les valeurs suivantes:

  • Téléphone: 514 111 1111
  • E-mail: 1@example.com
  • Rechercher: Numéro de passeport: AB123456
  • Commentaires: visa: 4916 0739.3667-6891, master:5428735149026050, SIN:123 123-123, ZIP:K2C 3N2, numéro de permis de conduire:P12345678912345, exemple de compte bancaire: 003-1234567
  • Nom complet: John Doe

Le formulaire affichera un message informant l'utilisateur qu'aucun renseignement personnel ne sera soumis.

Après l'envoi du formulaire, la sortie des paramètres d'url sera :?preferredNumber=&email=&search=Passport+Nr%3A+&comments=visa%3A+%2C+master%3A%2C+SIN%3A%2C+ZIP%3A%2C+Driving+Licence+Nr%3AP%2C+Bank+account+example%3A+&noScrubField=John+Doe
